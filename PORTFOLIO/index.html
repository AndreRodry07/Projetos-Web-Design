Sobre

Sou uma pessoa apaixonada por tecnologia e com habilidades em HTML, CSS e JavaScript. Sou um desenvolvedor em constante evolução, buscando aprimorar minhas competências e acompanhar as últimas tendências no campo da programação web.

Com meu conhecimento em HTML, sou capaz de criar a estrutura básica de uma página da web, definindo a hierarquia dos elementos e organizando o conteúdo de forma lógica. Utilizo as tags apropriadas para headings, parágrafos, listas, links e imagens, garantindo uma experiência de navegação intuitiva para os usuários.
